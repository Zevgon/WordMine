<h2> HTML endpoints </h2>
  * Root
    * `GET / ` triggers React components to load

<h2>  JSON endpoints </h2>
  * Users
    * `POST /api/users`
    * `GET /api/users`
  * Session
    * `POST /api/sessions`
    * `GET /api/sessions`
    * `DELETE /api/sessions`
  * Trees
    * `GET /api/trees`
    * `GET /api/trees/:treeId`
    * `GET /api/trees/:treeId`
    * `GET /api/choices`
    * `POST /api/trees_users`
  * Nodes
    * `GET /api/nodes/:nodeId`
    * `PATCH /api/nodes/:nodeId`
